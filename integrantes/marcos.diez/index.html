PROYECTO 1 - Juego de plataformas Escapa del Arbol

Proyecto realizado en Unity para la asignatura Proyectos II.

Se trata de un videojuego de plataformas con la posibilidad de atacar. El objetivo es llegar al final del recorrido, por el que te encontraras enemigos, obstaculos, pasadizos secretos y monedas. Fue relaizado en Unity, utilizando C#, sprites varios de Intenet.

PROYECTO 2 - Juego de Combate por turnos

Proyecto realizado con Java para la asignatura Diseño Software

Juego por turnos jugado por consola. Puedes elegir diferentes personajes, equipamiento y las acciones a elegir en cada combate. Cada uno tendrá un ataque, defensa, vida... diferentes. Tu objetivo es avanzar lo máximo posible por los diferentes mundos, eliminando a todos los enemigos que se te pongan por delante. Desarrollado al completo en Java utilizando Eclipse. Memoria del proyecto en la descarga.
